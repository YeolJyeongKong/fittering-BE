## 핏터링(fittering) Back-end
**핏터링**은 개인의 체형을 분석해 가장 fit한 옷들을 추천하고 필터링해서 보여주는 쇼핑몰 서비스입니다.<br>
[fittering-BE](https://github.com/YeolJyeongKong/fittering-BE)는 핏터링 서비스에서 **백엔드 코드를 관리합니다.**<br>
